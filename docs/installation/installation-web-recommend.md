# Groobee 스크립트 설치 가이드 - 추천 캠페인 사용법

이 문서는 웹 환경에서 Groobee 스크립트를 **실제 서비스에 맞게 설치**하기 위한 가이드 중 Groobee 추천 캠페인 사용 방법을 기술한 문서입니다.

---

## 사전 요구 사항

- (필수) 공통 스크립트가 설치되어 있어야 합니다.    
👉 [공통 스크립트 설치 가이드](../installation/installation-web-common-script.md)
- (권장) 회원 정보 연동이 되어 있어야 합니다.  
👉 [회원 정보 연동 가이드](../installation/installation-member-data.md)
- (권장) 행동 이력 수집 코드가 구현 되어 있어야 합니다.  
👉 [행동 이력 수집 가이드](../installation/installation-web-action.md)

> 회원 정보와 행동 이력이 수집되지 않으면, 추천 품질이 저하될 수 있습니다.
