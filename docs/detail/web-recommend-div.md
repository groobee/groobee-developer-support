# AI 상품 추천: DIV 형

## 개요
어드민에서 추천 템플릿 유형을 선택하고 에디터를 이용해 상세하게 디자인 할 수 있습니다. 별도 디자인 커스터마이징은 불가능합니다.

- **PC**
  - **1줄/2줄 가로형**
  - **세로형**
  - **1줄/2줄 슬라이드**
- **모바일**
  - **1줄/2줄 가로형**

## 연동 방식
추천 노출 위치에 아래와 같은 DIV 컨테이너를 배치하고, 추천 스크립트가 자동 렌더링합니다. Groobee 스크립트 실행 시점에 노출되며 노출 위치에 캠페인키를 ID를 가진 DIV 컨테이너가 존재해야 합니다.

```javascript
<div class="groobee_recommendation" id="추천캠페인키"></div>
```
