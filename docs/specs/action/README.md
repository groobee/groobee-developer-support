# 행동 유형
- 이 문서는 Groobee에서 사용되는 주요 행동 유형 목록을 설명합니다.

| 행동 이름 | 행동 코드 | 설명 |
|---|---|---|
| 메인 페이지 | MA | 메인 페이지 방문 시 수집 |
| 상품 상세 페이지 | VG | 상품 상세 페이지 방문 시 수집 |
| 장바구니 페이지 | VC | 장바구니 페이지 방문 시 수집 |
| 검색 결과 페이지 | SE | 검색 결과 페이지 방문 시 수집 |
| 주문서 페이지 | OR | 주문서 페이지 방문 시 수집 |
| 주문 완료 페이지 | PU | 주문 완료 시 수집 |
| 카테고리 페이지 | CA | 카테고리 페이지 방문 시 수집 |
| 회원가입 완료 페이지 | MJ | 회원가입 완료 시 수집 |
| 기타 페이지 | LO | 이벤트 등 기타 페이지 방문 시 수집 |

- 행동 유형에서 사용되는 데이터 스키마에 대한 설명은 👉 [Schema](./schema.md) 문서를 참고해주세요.

---

<a id="MA"></a>
## 메인 페이지 (MA)
- 메인 페이지 방문 시 수집하는 행동입니다.

<a id="VG"></a>
## 상품 상세 페이지 (VG)
- 상품 상세 페이지 방문 시 수집하는 행동입니다.
- 특정 상품의 상세 정보를 확인하는 페이지 방문 시 수집해야하는 행동입니다.

<a id="VC"></a>
## 장바구니 페이지 (VC)
- 사용자가 장바구니 목록을 **확인**하는 페이지 방문 시 수집하는 행동입니다.
- [장바구니 추가 이벤트(AC)](#AC) / [장바구니 제거 이벤트(DC)](#DC)와는 별도의 행동 유형입니다.

<a id="SE"></a>
## 검색 결과 페이지 (SE)
- 사용자가 검색어를 입력하고 결과를 확인한 경우 수집하는 행동입니다.

<a id="OR"></a>
## 주문서 페이지 (OR)
- 결제 진행 직전 단계인 주문서 페이지 방문 시 수집하는 행동입니다.

<a id="PU"></a>
## 주문 완료 페이지 (PU)
- 결제가 정상적으로 완료되었을 때 수집하는 행동입니다.
- 실제 매출이 발생한 시점에 수집해야하는 행동입니다.

<a id="CA"></a>
## 카테고리 페이지 (CA)
- 특정 상품 카테고리 페이지 방문 시 수집하는 행동입니다.

<a id="MJ"></a>
## 회원가입 완료 페이지 (MJ)
- 회원가입이 완료되었을 때 수집하는 행동입니다.

<a id="LO"></a>
## 기타 페이지 (LO)
- 이벤트, 기획전 등 특정 페이지 방문 시 수집하는 행동입니다.
- 일반 행동 유형에 포함되지 않는 경우 사용됩니다.

<a id="CE"></a>
## 커스텀 이벤트 (CE)
- 개발자가 정의한 특정 행동을 수동으로 수집합니다.
  - 예: 버튼 클릭, 특정 영역 노출, 영상 재생 완료 등
- 주요 활용 목적:
  - 서비스 맞춤 행동 분석
  - 세밀한 세그먼트 구성
  - 고도화된 개인화 전략 수립

<a id="AC"></a>
## 장바구니 담기 이벤트 (AC)
- 사용자가 상품을 장바구니에 추가했을 때 수집하는 행동입니다.

<a id="DC"></a>
## 장바구니 제거 이벤트 (DC)
- 사용자가 장바구니에서 상품을 삭제했을 때 수집하는 행동입니다.
