# Groobee 개발자 지원 문서

Groobee SDK 및 스크립트를 사용하는 **개발자와 서비스 운영자**를 위한 지원 문서입니다.
웹, iOS, Android 및 하이브리드 앱 환경에서 Groobee를 빠르고 안정적으로 연동하는 것을 목표로 합니다.
