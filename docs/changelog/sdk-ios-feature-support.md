# iOS SDK - 기능 지원 범위 (Feature Support)

이 문서는 Groobee iOS SDK에서 제공하는 기능의 지원 범위를
버전 기준으로 정리한 문서입니다.  
각 기능의 지원 시작 버전과 현재 상태를 확인할 수 있습니다.

---

## 기능 상태 정의

| 상태 | 설명 |
|------|------|
| ✅ 지원 | 정상적으로 사용 가능 |
| ⚠️ 제한적 지원 | 일부 조건 또는 환경에서 제약 존재 |
| 🧪 베타 | 정식 기능 전 단계 |
| 🗑 사용 중단 예정 | 향후 제거 예정 |
| ❌ 미지원 | 지원하지 않음 |

---

## 기능 지원 목록

| 기능 | 상태 | 지원 시작 버전 | 지원 종료 버전 | 비고 |
|------|------|----------------|----------------|------|
| SDK 초기화 | ✅ 지원 | 1.0.0 | - | - |
