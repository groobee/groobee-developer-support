# Android SDK - Feature Support

이 문서는 Groobee Android SDK의 기능 제공 범위를 버전 기준으로 정리합니다.  
각 기능의 지원 시작 버전과 상태를 확인할 수 있습니다.

## Feature Status 정의

| 상태 | 의미 |
|------|------|
| ✅ Supported | 정상 지원 |
| ⚠️ Limited | 일부 제약 조건 존재 |
| 🧪 Beta | 베타 기능 |
| 🗑 Deprecated | 제거 예정 |
| ❌ Not Supported | 미지원 |
